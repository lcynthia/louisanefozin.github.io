SWEET PATIENCE IS AN INTERACTIVE PROJECT THAT CREATES AN PSYCHEDELIC EXPERIENCE OF GROWING A TREE IN THE DESERT. THE PLANT GROWS USING THE USER'S SOUND INPUT AND REACTS TO DIFFERENT ACTIONS FROM THE USER. THIS WORK WAS DONE IN THE CONTEXT OF A PROGRAMMING CLASS IN COMPUTATION ARTS.

FR

SWEET PATIENCE EST UN PROJET INTERACTIF QUI TENTE DE CREER UNE EXPERIENCE PSYCHEDELIQUE D'UN ARBRE QUI POUSSE DANS LE DESERT. LA PLANTE GRANDIT EN UTILISANT LES DONNEES SONORES ET EN REAGISSANT AUX DIFFERENTES ACTIONS PRODUITES PAR L'UTILISATEUR. CE TRAVAIL A ETE EXECUTE DANS LE CONTEXTE D'UN COURS DE PROGRAMMATION EN ARTS NUMERIQUES.
